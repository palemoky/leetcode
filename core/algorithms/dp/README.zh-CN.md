
动态规划（DP）是一种将复杂问题分解为子问题，通过保存子问题的最优解，递推得到全局最优解的算法思想。其核心在于状态定义、状态转移方程和子问题重叠。    

DP五部曲（建立思维框架）：
1. 定义 dp 数组的含义。
2. 找出状态转移方程。
3. 初始化 dp 数组。
4. 确定遍历顺序。
5. 举例推导 dp 数组。

核心题型（每个至少做一道）：
- 背包问题： LeetCode 416. Partition Equal Subset Sum (0-1背包的变体)
- 路径问题： LeetCode 62. Unique Paths
- 序列问题： LeetCode 300. Longest Increasing Subsequence (最长递增子序列)
